# AP09_Play_Game
Seguridad informática modulo MF0487 
## 1. Identificación de la actividad
- Módulo: MF0487_3 - Auditoría de seguridad informática
- Actividad: AP09 Play Game
- Autor: Andrés Campo Pereddo
## 2. Objetivo
Desarrollar una función capaz de construir rectángulos mediante
caracteres definidos, utilizando argumentos recibidos desde la
línea de comandos.
## 3. Archivo entregado
- PG1.py: ejercicio principal.
## 4. Requisitos técnicos
- Python 3.
- No requiere bibliotecas externas.
- Utiliza únicamente el módulo estándar `sys`.
## 5. Ejecución de PG1.py

```bash
python PG1.py 5 3
```

Resultado esperado
```text
o---o
|   |
o---o
```

Otras pruebas:

```bash
python PG1.py 5 1
python PG1.py 1 1
python PG1.py 1 5
```
